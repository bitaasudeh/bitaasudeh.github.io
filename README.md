# Personal Portfolio Website

This is my personal portfolio website.

## Automatic Deployment

This website is set up for automatic deployment to GitHub Pages. After pushing to the `main` branch, the website will automatically update.

### Setup Instructions

1. Go to your repository on GitHub
2. Click **Settings** â†’ **Pages**
3. Under **Source**, select **GitHub Actions**
4. The workflow will automatically deploy your site whenever you push to the `main` branch

Your website will be available at: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`
