Under Review

[SIGMOD 2026] Unbiased Binning: Fairness-aware Attribute Representation
Abolfazl Asudeh, Zeinab Asoodeh, Bita Asoodeh, Omid Asudeh
Abstract: Discretizing raw features into bucketized attribute representations is a popular step before sharing a dataset. We address discretization bias by introducing the unbiased binning problem that finds the closest discretization to equal-size binning that satisfies group parity across different buckets. We develop efficient dynamic programming and local search algorithms for optimal solutions.
